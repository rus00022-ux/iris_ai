# IRIS AI — интерактив «Анализ подписи»

## Быстрый старт локального сервера

1. Убедитесь, что на компьютере установлен Node.js версии 18 или новее.
2. В корне проекта выполните команду:

   ```bash
   npm start
   ```

   По умолчанию сервер запустится на адресе `http://0.0.0.0:8000` и будет доступен всем устройствам в вашей локальной сети.

3. Узнайте IP-адрес компьютера в локальной сети (например, `192.168.0.12`).
4. Откройте на iPad браузер Safari и перейдите по ссылке `http://<ваш_ip>:8000/interactive.html`.

## Полезные опции

Сервер поддерживает параметры запуска:

```bash
node server.js --host 0.0.0.0 --port 9000
```

- `--host` — адрес, к которому привязывается сервер (по умолчанию `0.0.0.0`).
- `--port` — порт для подключения (по умолчанию `8000`).

После запуска также доступна админ-панель по адресу `http://<ваш_ip>:8000/admin.html`.
